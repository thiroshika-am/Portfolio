# Developer Portfolio

A premium, terminal-inspired developer portfolio designed to present clean, structured software engineering achievements.

## Tech Stack
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion

## Features
- Interactive node/grid canvas background.
- Terminal-inspired typing prompts.
- Categorized skills panel.
- Responsive, premium design.
